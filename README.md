#EventFacilitationSystem
A Django-based Event Management System for schools and organizations.
## Features

### Admin
- Create and approve events
- Allocate venues/resources
- View and edit (event,registrations)

### Teacher
- create events
- Manage participant lists
- Upload event materials

### Student
- View upcoming events
- Register/unregister for events
- Receive notifications(email)
- Submit feedback

### Parent
- Receive event notification (email)
- Provide feedback on event success
